Code Challenge: Virtual Party

Scenario

You are hosting a virtual party, and you want to ensure that your guests have an amazing time. You'll need to send out invitations, prepare activities, open the virtual dance floor, give a host speech, and even surprise your guests with a special DJ appearance!

In this exercise, you will complete the JavaScript functions to create a simulated virtual party experience using both synchronous and asynchronous programming.

Instructions

Task 1: Send Invitations

  1. Complete the function named sendInvitations.
  2. Inside the function, use a for...of loop to iterate over the array of guests.
  3. Inside the loop, log a message to invite each guest.
  4. In your console, call the sendInvitations function to test it, and make any corrections if necessary.

Task 2: Prepare Party Activities

  1. Complete the function named prepareActivities.
  2. Inside the function, use setTimeout to simulate the time it takes to prepare the activities (2 seconds).
  3. Inside the setTimeout callback, log a message announcing that the activities are ready.
  4. Call the prepareActivities function to test it in your console.

Task 3: Virtual Dance Floor

  1. Complete the function named virtualDanceFloor.
  2. Inside the function, use setInterval to periodically log a message inviting guests to dance (every 1 second).
  3. Inside the setInterval callback, stop after logging 5 messages.
  4. Call the virtualDanceFloor function to test it in your console.

Task 4: Host Speech

  1. Complete the function named hostSpeech.
  2. Inside the function, log a welcoming message immediately.
  3. Use setTimeout to log a second message after a delay (1.5 seconds).
  4. Call the hostSpeech function to test it in your console.

Challenge Task: Surprise Guest

  1. Complete the function named surpriseGuest.
  2. Inside the function, use setTimeout to log a message announcing the surprise guest after a delay (3 seconds).
  3. Call the surpriseGuest function to test it in your console.

Launch Party Event

  1. Complete the function named launchParty that calls all the party functions in the proper sequence.
  2. Create an event listener for the "Launch Party!" button that calls the launchParty function when clicked.
  3. Click the button to start your party!
